Single-page personal portfolio website for **Suracha "Pom Pam" Hanthongchai** ‚Äî Software Developer & Computer Educator (healthcare software, scalable apps, AI-integrated workflows). Fully bilingual **Thai/English**; one dark theme; deploys as pure static files.

## Stack

- **Vite 8 + React 19 + TypeScript (strict)** ‚Äî SPA, no router, no SSR
- **Tailwind CSS v4** via `@tailwindcss/vite` ‚Äî CSS-first config (`@theme` in `src/index.css`; there is **no** `tailwind.config.js`)
- **framer-motion 12** ‚Äî reveals via `whileInView`; global `<MotionConfig reducedMotion="user">`
- **i18next + react-i18next + i18next-browser-languagedetector** ‚Äî locales in `src/i18n/locales/{en,th}.json`; detection localStorage ‚Üí navigator; fallback `en`
- **Vitest + jsdom + Testing Library** ‚Äî setup file stubs `IntersectionObserver`, `ResizeObserver`, `matchMedia`
- **ESLint flat config + `tsc --noEmit`** ‚Äî zero warnings allowed
- Fonts self-hosted: `@fontsource/prompt` (headings) + `@fontsource/sarabun` (body), Thai + Latin subsets

## i18n Workflow (critical)

- **Never hardcode user-visible text in components.** Every string renders via `useTranslation()` ‚Üí `t('section.key')`.
- Adding text = add the key to **BOTH** `src/i18n/locales/en.json` **and** `th.json`. A parity test fails the suite if the key sets differ or any value is empty.
- Data modules (`src/data/*`) hold ids/images/links only; their display text lives under keys like `projects.items.<id>.title`.
- `document.documentElement.lang` is synced automatically by `src/i18n/index.ts` ‚Äî don't set it manually.
- Thai text rules: no `tracking-*` (letter-spacing) or uppercase transforms on Thai strings; keep `leading-relaxed` line-height.

## Swapping Real Content

- **Images**: replace files in `public/images/` keeping the same filenames (`profile.svg`, `project-ehp.svg`, `cert-arduino.svg`, `gallery-01.svg`, ...). Any raster format works if you also update the path in the matching `src/data/*` entry.
- **Resume**: overwrite `public/resume.pdf` (placeholder is generated by `node scripts/make-resume.mjs`).
- **Identity/links**: edit `src/config/site.ts` (email, GitHub, LinkedIn, resume path).
- **Thai name spelling** in `th.json` is an unverified transliteration ‚Äî confirm with the owner.
